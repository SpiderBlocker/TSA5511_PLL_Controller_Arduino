TSA5511 PLL Controller for Arduino

Description

PLL controller for the TSA5511, initially intended to replace the proprietary controller for the DRFS06 exciter by Dutch RF Shop, but it can be used for any other TSA5511 based exciter, operating in a VCO frequency range of 64 MHz up to 1,300 MHz as per specification of the TSA5511. It features an intuitive menu interface for making various system settings as explained in detail below. All configurable settings are stored in EEPROM and automatically recalled upon restart.

Hardware

  • Arduino Nano or compatible
  • Standard 16x2 LCD display (4-bit mode) with backlight and contrast adjustment
  • Three pushbuttons (DOWN/SET/UP), each with a 470 nF debouncing capacitor
  • Optional PLL lock LED, also acts as blinking fault indicator
  • LCD backlight control via PWM (default pin 6)
  • Pull-up resistors required on SDA/SCL; 1–2 kΩ recommended if RF decoupling is used
  • Recommended to power controller separately from TSA5511 when used with DRFS06

Usage

  • Double-click SET to open the main configuration menu. Options include:
    • VCO SUBMENU
      • VCO FREQ. BAND – Select from predefined bands. Each remembers last used frequency.
      • FREQ. PRECISION – Set VCO frequency decimal precision. Lower precision will round current frequency accordingly. Max resolution: 3 decimals @ 1.6 MHz XTAL, 2 decimals @ 3.2 MHz XTAL.
      • EXIT SUBMENU – Return to main menu.
    • PLL SUBMENU
      • PLL CHARGE PUMP – Set locked-state current (HIGH = 220 µA, LOW = 50 µA). Use HIGH for DRFS06.
      • PLL XTAL FREQ. – Select 1.6 MHz or 3.2 MHz XTAL. Affects maximum VCO range. 1.6 MHz is unofficially supported but works reliably.
      • EXIT SUBMENU – Return to main menu.
    • STATION NAME – Edit station name displayed when PLL is locked. Use UP/DOWN/SET. Auto-scroll supported.
    • BACKLIGHT DIMMER – Toggle auto-dimming of the LCD backlight.
    • EXIT MENU
      • SAVE & EXIT – Save changes to EEPROM and return to main screen.
      • DISCARD & EXIT – Ignore changes and return to main screen.
      • CANCEL – Return to first main menu option.
  • VCO frequency can be adjusted with UP/DOWN and confirmed with SET. Inactivity reverts to main screen without changes.
  • Hold UP/DOWN to auto-scroll with acceleration.
  • Backlight dims automatically in idle mode (PLL locked). Hold SET to turn off completely. Press any key to restore.
  • On I²C error: check wiring and press SET to retry. Multiple retries occur before showing an error.
