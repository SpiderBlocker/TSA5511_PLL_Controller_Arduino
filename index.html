TSA5511 PLL Controller for Arduino – FM Transmitter Project

About the Project

Develop an Arduino-based controller for the TSA5511 PLL, designed to replace the proprietary controller for the DRFS06 exciter by Dutch RF Shop. Suitable for any TSA5511-based FM transmitter or exciter.

Features include robust I²C communication, EEPROM storage for settings, and LCD backlight control.

Technical Specifications

  • Compatible with Arduino Nano or similar boards.
  • Uses a standard 16x2 LCD display.
  • Three-button interface for easy control.
  • PLL frequency range: 64 MHz - 1.3 GHz.
  • Stores frequency and settings in EEPROM.
  • Built-in LCD backlight dimmer.

How to Use

  • Configure the XTAL frequency and band edges in the code.
  • Press and hold SET during startup to enter station name editing mode.
  • Adjust frequency using UP/DOWN buttons.
  • Backlight dims after inactivity; double-click SET to disable dimming.
  • For I²C errors, verify hardware and restart with SET.
